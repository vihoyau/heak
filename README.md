## 绘制学习路线图
```seq
挖掘漏洞->抓包分析: 使用burpsuit工具响应断点，更改请求数据，字典暴力登录等
China-->Andrew: How are you? 
Andrew->>China: I am good thanks!
```
```flow
st=>start: 用户登陆
op=>operation: 登陆操作
cond=>condition: 登陆成功 Yes or No?
e=>end: 进入后台
st->op->cond
cond(yes)->e
cond(no)->op
```
